# receipt-OCR

### Prerequisites
```
Python 3
pip
virtualenv
```
### Installation
Create a virtualenv and activate it, then:
```
pip install -r requirements.txt
```
### Demo
Images will be saved in the img folder, to run it use:
`python main.py <image_name>`

```
python main.py 1.jpg
```
